Intel Image Classification¶

The intel image classification problem concerned predicting the class of images grouped into six classes (buildings, forest, glacier, mountain, sea and street). We will initially train a simple neural network with a single 2D convoutional layer and use training imags without augmentation before training the model using the ResNet50 pre-trained model and training data with augmented images.

Analyse Model Performance on Test Set¶

def pred_results(pred, train_generator):
    predicted_class_indices=np.argmax(pred,axis=1)
    labels = (train_generator.class_indices)
    labels = dict((v,k) for k,v in labels.items())
    predictions = [labels[k] for k in predicted_class_indices]
    filenames=test_generator.filenames
    results=pd.DataFrame({"Filename":filenames,
                          "Predictions":predictions})
    get_actuals = lambda x : x['Filename'].split('/')[0]
    results['actuals'] = results.apply(get_actuals, axis=1)
    results['correct'] = results.apply(lambda x: 1 if x['Predictions'] == x['actuals'] else 0, axis=1)
    correct = results['correct'].sum()
    total = results['Filename'].count()
    success_rate = correct/total
    return success_rate, results

Conclusion¶

Using ResNet50 architecture with image augmentation enabled an improvement from 71.4% to 90.5% in accuracy on the test set. Looking at the images that were not classified correctly in the test set, it's clear that many of these contain elements from other classes such as the above image that had a building label but contains aspects of a street in the image that was obviously what the model used in it's classification.

For future iterations of the model we can look to try some or all of the following to improve accuracy:

  • Tune the learning rate
  • Try different image augmentation
  • Try different pre-trained models with different layers frozen
